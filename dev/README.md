# Ferramentas de Desenvolvimento

Arquivos de desenvolvimento e debugging não utilizados em produção.

## Arquivos

- `teste-api.html` - Interface para testar API do otimizador
- `diagnostico-maps.html` - Ferramenta de debug do Google Maps
- `Prototipo.html` - Protótipo original do sistema
- `dashboard.html` - Dashboard incompleto (feature futura)

## Uso

Abrir diretamente no navegador com servidor local:

```bash
npm run dev
```

Acessar:
- http://localhost:8080/dev/teste-api.html
- http://localhost:8080/dev/diagnostico-maps.html
- http://localhost:8080/dev/Prototipo.html
- http://localhost:8080/dev/dashboard.html

## Nota

Estes arquivos são mantidos para propósitos de desenvolvimento e debugging, mas não fazem parte do fluxo de produção.
